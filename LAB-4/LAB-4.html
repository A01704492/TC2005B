¿Qué diferencias y semejanzas hay entre Java y JavaScript?


Diferencias

Propósito Java tiene como proposito desarrollar aplicaciones de escritorio, móviles y empresariales. Mientras que JavaScript se utiliza para desarrollar aplicaciones web incluyendo front-end y back-end.
Tipado Java está fuertemente tipado mientras que JavaScript está débilmente tipado.
Compilación Mientras que Java es compilado, JavaScript es interepretado.
Herencia Java permite herencia múltiple mientras que JavaScript no.
Sintaxis JavaScript utiliza opcionalmente llaves '{}' y el punto y coma ';' opcionalmente, para Java estos se utilizan para delimitar bloques de código y forzosamente al final de cada sentencia.


Similitudes

Programación orientada a objetos Ambos son lenguajes de programación orientados a objetos.
C Sintaxis similar a C.
Tipos de Datos Ambos lenguajes tienen tipos de datos básicos similares, como números, cadenas, booleanos, etc.
DOM Capacidad de manipular el DOM (Document Object Model) y manejar eventos en el navegador.
Variedad Ambos lenguajes tienen una amplia variedad de bibliotecas y frameworks para facilitar el desarrollo de aplicaciones.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?


Confusión de variables
Puede causar confusión y errores si se utiliza el mismo nombre de variable global dentro de una función.


Interferencia con otras partes del código
Se corre el riesgo de que la variable se utilice o se modfique accidentalmente en otra parte del código.



Rendimiento
Son menos eficientes las variables globales que las locales debido a que deben de ser buscadas en todo el ámbito del programa cada vez que se utilizan.


Mantenibilidad
El uso excesivo de variables globales hará que el código sea más difícil de entender, mantener y depurar.


Seguridad
Las variables globales pueden ser modificadas desde cualquier parte del programa, lo que presenta un riesgo si se utilia para almacenar información sensible o privada.
